# Vivop Plus Prototype

Este es un prototipo básico de la app Vivop Plus en Flutter.

## Funcionalidades:
- Login con cuenta de admin (`admin` / `adminpeladoreligious`)
- Vista de admin y cliente
- Pantalla inicial

## Compilación:
1. Asegúrate de tener Flutter instalado.
2. Abre el proyecto en tu editor.
3. Ejecuta `flutter pub get`.
4. Corre con `flutter run`.

*Nota: Aún falta integración con Google Drive y reproducción de video.*
